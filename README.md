# umi-ts-electron
Electron app template via Umi 


### How to run:

1. Clone

```shell
$git clone https://github.com/skitsanos/umi-ts-electron.git
```

2. Change directory 

```shell
$cd umi-ts-electron
```

3. Install dependencies (yarn or npm, whatever you prefer, although npm is slower)

```shell
$yarn install
```

4. Run

```shell
$yarn run electron:dev
```
